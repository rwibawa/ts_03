# ts_02
* [Debugging typescript with vscode](https://code.visualstudio.com/docs/typescript/typescript-debugging)

## 1. Setup
```bash
$ mkdir ts_02
$ cd ts_02
$ npm i -g typescript
$ tsc init
$ code .
```

## 2. Run
[Ctrl-Shift-B] to build.
[Ctrl-Shift-D] to go to the Debugger.
